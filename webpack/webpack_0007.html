    # 基本结构

    ast语法树名词解释

    type：当前节点的类型

    start：当前节点的起始位

    end：当前节点的末尾

    range：当前节点的起止位数组

    sourceType：源码类型

    {
  "type": "Program",
  "start": 0,
  "end": 12,
  "range": [0, 12],
  "body": [],
  "sourceType": "module"
}

    # ast节点对照表

    类型原名称 序号 中文名称 描述
    Program 1 程序主体 整段代码的主体
    VariableDeclaration 2 变量声明 声明一个变量，例如 var let const
    FunctionDeclaration 3 函数声明 声明一个函数，例如 function
    ExpressionStatement 4 表达式语句 通常是调用一个函数，例如 console.log()
    BlockStatement 5 块语句 包裹在 {} 块内的代码，例如 if (condition){var a = 1;}
    BreakStatement 6 中断语句 通常指 break
    ContinueStatement 7 持续语句 通常指 continue
    ReturnStatement 8 返回语句 通常指 return
    SwitchStatement 9 Switch 语句 通常指 Switch Case 语句中的 Switch
    IfStatement 10 If 控制流语句 控制流语句，通常指 if(condition){}else{}
    Identifier 11 标识符 标识，例如声明变量时 var identi = 5 中的 identi
    CallExpression 12 调用表达式 通常指调用一个函数，例如 console.log()
    BinaryExpression 13 二进制表达式 通常指运算，例如 1+2
    MemberExpression 14 成员表达式 通常指调用对象的成员，例如 console 对象的 log 成员
    ArrayExpression 15 数组表达式 通常指一个数组，例如 [1, 3, 5]
    NewExpression 16 New 表达式 通常指使用 New 关键词
    AssignmentExpression 17 赋值表达式 通常指将函数的返回值赋值给变量
    UpdateExpression 18 更新表达式 通常指更新成员值，例如 i++
    Literal 19 字面量 字面量
    BooleanLiteral 20 布尔型字面量 布尔值，例如 true false
    NumericLiteral 21 数字型字面量 数字，例如 100
    StringLiteral 22 字符型字面量 字符串，例如 vansenb
    SwitchCase 23 Case 语句 通常指 Switch 语句中的 Case
